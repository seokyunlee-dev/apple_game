# 프로젝트 개요: '사과 게임(Apple Game)' 자동화 봇 개발

**목표:** 화면 내의 숫자들을 인식하여 합이 10이 되는 조합을 찾아 자동으로 마우스를 드래그하는 '사과 게임' 오토 플레이 파이썬(Python) 프로그램을 작성해 줘. 

## 🎮 게임 룰
- 1부터 9까지의 숫자가 적힌 사과들이 격자(Grid) 형태로 배치되어 있음.
- 가로, 세로, 혹은 직사각형 형태로 드래그하여 선택한 사과들의 숫자 합이 정확히 '10'이 되면 해당 사과들이 사라짐.
- 최대한 많은 사과를 없애는 것이 목표.

## 🛠️ 핵심 구현 요구사항

**1. 영역 선택 기능 (Trigger & UI)**
- 사용자가 `Ctrl + Shift` 키를 누르면 화면 캡처 모드가 실행되어야 함.
- 마우스 드래그를 통해 게임이 진행 중인 화면 영역(ROI)을 지정할 수 있는 반투명 오버레이 UI가 필요함 (예: Tkinter, PyQt 또는 Snipping Tool 방식).

**2. 화면 캡처 및 숫자 인식 (Vision & OCR)**
- 지정된 영역을 캡처한 뒤, 이미지 처리(OpenCV 등)를 통해 사과들이 배치된 M x N 형태의 격자(Grid)를 파악해야 함.
- 각 격자 칸 안의 숫자(1~9)를 정확하게 추출하여 2차원 배열(List) 형태로 매핑해 줘 (pytesseract, easyocr 또는 템플릿 매칭 활용).
- 이미 없어져서 비어있는 칸은 0 또는 None으로 처리.

**3. 알고리즘 (Algorithm)**
- 2차원 배열 안에서 인접한 칸들(1xN, Nx1, 가로세로 직사각형 형태)의 합이 '10'이 되는 모든 경우의 수를 찾는 알고리즘을 작성해 줘.
- (주의) 드래그는 반드시 직사각형 형태(선 또는 면)로만 가능하며, 대각선이나 불규칙한 모양은 불가능함.

**4. 마우스 자동 제어 (Automation)**
- 합이 10이 되는 조합을 찾았다면, 해당 조합의 시작 좌표에서 끝 좌표까지 자동으로 마우스를 이동시키고 드래그(Mouse Down -> Move -> Mouse Up)하는 기능을 구현해 줘 (예: PyAutoGUI 사용).
- 사람의 조작처럼 보이기 위해 드래그 속도에 약간의 딜레이를 주거나, 한 콤보를 맞춘 후 짧은 대기 시간을 부여해 줘.

## 💻 추천 기술 스택 (Agent, 네가 판단해서 최적의 라이브러리를 사용해도 좋아)
- Language: Python
- Hotkey & Input: `keyboard`, `pynput`
- GUI (영역 지정): `tkinter`, `PyQt5`
- Vision: `opencv-python`, `pytesseract` (or `easyocr`)
- Automation: `pyautogui`

**요청:** 먼저 전체적인 아키텍처와 사용할 라이브러리들을 제안해 주고, 가장 먼저 구현해야 할 '영역 선택 및 캡처(1번)' 기능의 코드부터 작성해 줘.
